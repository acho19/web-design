About Health Code Violations

If a restaurant is given a health code violation, the restaurant on this database on NYC Open Data receives three important factors: Violation Code, Violation Description, and Crtical Flag. The Violation Code organizes the violation in a numerical sense, the Violation Description describes the found violations in a restaurants and the Critical Flag determines if the violation is critical or not critical. The New York Department of Health and Mental Hygiene describes critical violations as one that can most likely cause food borne illnesses.
